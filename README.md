# ¿Que es Java Script?
    Es un Lenguaje Interpretado, osea que todo el codigo que escribimos no tiene que pasar por un compilador, si no que va directo a ejecutarse al navegador, que tiene una herramienta llamada interpreta, agarrando todo el codigo y lo traduce en tiempo real.
    Es un lenguaje Orientado a Objetos.
    Es un lenguaje Imperativo, que va ejecutandose paso por paso.
    Es un Lenguaje Case Sensitive, es sensible a mayusculas y minusculas.
    Es basado en Prototipos, Instancias y Classless.
    Es de Tipado Debil, el valor de las variables puede cambiar a lo largo del tiempo, es mas flexible para cambiar entre tipos de datos.
    Es un Lenguaje Dinamico, la variable no se ajusta al dato, el dato se ajusta a la variable.
    Java Script esta construida sobre otras tegnologias, ECMASCRIPT (5.1 Y 6, 7, 8, Next)

# Variables

    Primero ¿Que es una variable? Es un espacio que guardamos en memoria, para usarlo devuelta.
    Tipos de Datos: String (cadena de Texto), Number, Boolean, Symbol.
        String: lo podemos escribir entre "" (Comillas dobles) o '' (Comillas simples).
        Number: Se escribe tal y como es.
        Booleano: se escribe 1 o 0, lo traducimos a true o false.
    Casos Especiales: undefined, null, nan. Los tres tipos de datos nos hablan de que la variable no esta definida o hay un error.
    Tipos de Variables: Let, Var y Const.
        CONST se tiene que declarar e iniciarlizar al mismo tiempo.
        LET y VAR se pueden iniciar por su parte y modificar por su parte.
        Let nos limita el alcance de la variable al bloque en el que estamos.
    Scope es el ambito de las Variables el contexto, por ejemplo el alcance de la variable se limita a ese bloque, por fuera del bloque esa variable no existe.
    Se pueden crear multiples variables separandolas por coma e inicializarlas en la misma linea.
    Hoisting se trata de las fases de creacion y de ejecucion.

# Operadores Intermedio

    Operadores de Comparacion comparan dos expresiones y devuelven un valor Boolean (true o false) que representa la relacion de sus valores.
    Operadores Logicos se basan en true y false para devolver algo, nos devuelven un resultado a apartir de que se cumpla (o no) una condicion, su resultado es booleano y sus operandos son valores logicos o asimilables a ellos.

# Condicionales 

    Los condicionales son estructuras de control que permiten tomar decisiones lógicas en el código. Evalúan una condición (si es verdadera o falsa) y ejecutan diferentes bloques de código según el resultado, permitiendo que tu programa sea dinámico e interactivo.
