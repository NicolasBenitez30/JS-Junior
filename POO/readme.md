# Programacion Orientada a Objetos

Definicion: Es un Paradigma de la programacion

# Conceptos Basicos

Clase: Es una fabrica de objetos, que crea un objeto con ciertas cualidades que le asignamos.
Objeto:
Atributo: Las caracteristicas de nuestro Objeto.
Metodo: Son las cosas que puede hacer nuestro Objeto.
Constructor: Es una Funcion Obligatoria de la clase, nos permite construir las propiedades del objeto.
Instanciacion: Por ultimo podemos decir que la clase esta "Instanciada".

# Caracteristicas

Abstraccion: Se intenta reducir la complejidad del objeto, lo mas reducido posible.
Modularidad: Es la capacidad de resolver un problema por partes, Se resuelve parte por parte hasta que crea una solucion para el probleam entero.
Encapsulamiento: Mantiene todos los datos privados, asi no se pueda acceder de manera sencilla.
Polimorfismo: Consiste en ver como un objeto se comporta de manera distinta ante el mismo metodo. Es la capacidad que tiene un objeto para comportarse distinto por sus propiedades.

# Otros Conceptos

Herencia:
Metodos Estaticos: Es un metodo que no necesita que la clase lo defina para poder ser creado.
Metodos Accesores (Getters, Setters)
    Setters: Son para Modificarlo o definirlo.
    Getters: Son para obtener un valor.
